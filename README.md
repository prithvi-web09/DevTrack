🚀 DevTrack

> **A Premium Developer Progress Dashboard built to help developers stay consistent, track growth, and showcase their journey.**

## ✨ Overview

DevTrack is a modern productivity dashboard designed specifically for developers.

Instead of scattering progress across notebooks, sticky notes, and multiple apps, DevTrack brings everything together into one elegant interface where you can monitor your learning journey.

Whether you're learning new technologies, building projects, or improving your coding consistency, DevTrack helps visualize your progress every day.

---

## 🎯 Features

### 📚 Study Tracker
- Log daily study hours
- Increment & decrement study sessions
- Data persists after refresh using Local Storage

### 🚀 Skill Arsenal
- Add new technical skills
- Set mastery level
- Interactive progress indicators
- Track learning growth visually

### 💻 Project Manager
- Create unlimited projects
- Add project status
- Write project descriptions
- Attach GitHub repository links
- Open repositories directly from the dashboard

### 🔥 Daily Motivation
- Streak tracker
- Productivity widgets
- Developer-focused UI

### 💾 Persistent Storage
- Uses Local Storage
- Progress remains even after refreshing the browser
